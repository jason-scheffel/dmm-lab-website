DNA-stabilized silver clusters (Ag-DNAs) are novel fluorophores that are finding numerous applications in nanophotonics, chemical sensing, and bioimaging. The fluorescence colors of Ag-DNAs can be tuned from bluegreen into the near-infrared by selecting the sequence of the single-stranded DNA that templates the cluster.

Using a training set of DNA template strands and the fluorescence spectrum associated with each strand, we mine discriminative multi-base DNA motifs that correlate with fluorescent cluster brightness. Furthermore, using such motifs to parameterize DNA templates, we develop a machine learning-based tool to design novel DNA templates that stabilize brightly fluorescent Ag-DNAs.
